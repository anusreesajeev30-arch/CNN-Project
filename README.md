Rice Type Classification using CNN

Description

This project uses a Convolutional Neural Network (CNN) to classify different types of rice (such as Basmati and other varieties) from images.

Technologies Used

- Python
- TensorFlow / Keras

Project Files

- "splitting_rice_dataset.ipynb" - Dataset splitting
- "Rice.ipynb" - Model training
- "rices.py" - Python implementation

How to Run

1. Run "splitting_rice_dataset.ipynb"
2. Run "Rice.ipynb" or "rices.py"

Model

Trained model file is not included due to large size (378MB).
You can retrain the model using the provided code.
